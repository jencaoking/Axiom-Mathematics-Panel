## 📝 快速示例

### 示例1: 绘制抛物线 (30秒)

1. 确保类型选择 "**显函数 y = f(x)**"
2. 在输入框中输入: `x^2`
3. 点击 **绘制函数** 按钮

✅ 看到抛物线了吗?

---

### 示例2: 参数化正弦波 (1分钟)

1. 输入: `A*sin(omega*x + phi)`
2. 点击 **绘制函数**
3. 观察下方自动生成的3个滑块:
   - **A** (振幅)
   - **omega** (角频率)
   - **phi** (相位)
4. 拖动 **A** 滑块从 1 到 3
   - ✅ 图象上下拉伸
5. 拖动 **omega** 滑块从 1 到 0.5
   - ✅ 周期变长(波形变宽)
6. 拖动 **phi** 滑块
   - ✅ 图象左右移动

🎯 **学习点**: 直观理解三角函数参数的几何意义!

---

### 示例3: 函数变换 (1分钟)

1. 输入基础函数: `x^2`
2. 点击 **绘制函数**
3. 依次点击变换按钮:
   - **右移 →** (1次)
   - **↑ 上移** (2次)
   - **Y轴伸缩** (1次)
4. 观察表达式变为: `1.5*((x-1)^2) + 2`
5. 对比原始抛物线和变换后的位置

🎯 **学习点**: 掌握函数平移和伸缩的组合效果!

---

### 示例4: 使用预设模板 (30秒)

1. 找到 "**预设模板**" 下拉菜单
2. 选择 "**正弦函数 A*sin(ωx+φ)**"
3. ✅ 自动填充表达式并绘制
4. 调整滑块探索

💡 **提示**: 模板是快速开始的好方法!

```

log(x)       对数 ln(x)

```

### 参数化函数
```
A*sin(omega*x + phi)    正弦波(3参数)
a*x^2 + b*x + c         二次函数(3参数)
a*exp(b*x)              指数函数(2参数)
a/(x-b) + c             反比例平移(3参数)
A*cos(x) + B*sin(x)     线性组合(2参数)

y^2 - 4*p*x             抛物线
```

## 💡 使用技巧

### 技巧1: 组合变换
连续点击多个变换按钮可以创建复杂图象:
```
x^2 
→ 右移 
→ Y轴伸缩 
→ 上移
= 1.5*(x-1)^2 + 1
```

### 技巧2: 参数+变换
先用滑块微调,再用变换按钮:
```
1. 输入: A*sin(omega*x)
2. 设置: A=2, omega=0.5
3. 点击: 左移 2 次
结果: 2*sin(0.5*(x+2))
```

### 技巧3: 多函数对比
可以同时绘制多个函数:
```
1. 绘制 x^2
2. 再绘制 (x-2)^2
3. 观察两条抛物线的位置关系
```

### 技巧4: 缩放查看
- 鼠标滚轮: 缩放画布
- 按住中键拖动: 平移画布
- 适合查看函数细节

## ❓ 常见问题

### Q: 为什么我的函数没有显示?
**A**: 检查以下几点:
- 表达式语法是否正确(使用 `*` 表示乘法)
- 函数名是否小写(`sin` 而非 `Sin`)
- 括号是否匹配
- 函数是否在可视范围内有定义

### Q: 滑块不响应怎么办?
**A**: 
- 确认表达式中包含参数字母
- 重新点击"绘制函数"
- 检查参数名是否与已知变量冲突(x, y, theta)

### Q: 如何清除所有函数?
**A**: 
- 编辑 → 删除(选中对象后)
- 或使用控制台: `engine.clear()`

### Q: 可以保存函数配置吗?
**A**: 
- 当前版本暂不支持
- 可以记录表达式和参数值
- 未来版本会添加此功能

   - 选择模板 "正弦函数"

   - 固定 omega=1, phi=0
   - 调整 A 从 1→2→3→0.5

   - 固定 A=1, phi=0
   - 调整 omega 从 1→2→0.5
   - 提问: "omega 与周期有什么关系?"
   
4. **探索相位** (3分钟)
   - 固定 A=1, omega=1
   - 调整 phi
   - 提问: "phi 如何影响图象位置?"
   
5. **总结** (2分钟)
   - 学生归纳三个参数的作用
   - 联系实际应用(声波、光波等)
