# 动态函数探索器使用指南

## 功能概述

动态函数探索器是 MathLab 新增的强大工具,用于交互式探索数学函数的性质和变换。主要功能包括:

1. **函数绘图**: 支持显函数、隐函数和极坐标函数
2. **参数滑块**: 实时调整函数参数,观察图象变化
3. **图象变换**: 一键应用平移、伸缩、反射等变换
4. **预设模板**: 快速加载常见函数类型

## 启动方式

1. 在菜单栏选择 **视图** → **函数探索器**
2. 或使用快捷键(如已配置)

面板将出现在左侧停靠区域。

## 基本使用

### 1. 绘制函数

#### 显函数 y = f(x)
- 选择函数类型为 "显函数 y = f(x)"
- 在输入框中输入表达式,例如:
  - `x^2` (抛物线)
  - `sin(x)` (正弦函数)
  - `2*x + 1` (直线)
  - `exp(x)` (指数函数)
- 点击 **绘制函数** 按钮或按 Enter 键

#### 隐函数 f(x,y) = 0
- 选择函数类型为 "隐函数 f(x,y) = 0"
- 输入表达式,例如:
  - `x^2 + y^2 - 1` (单位圆)
  - `x*y - 1` (双曲线)
- 点击 **绘制函数**

#### 极坐标 r = f(θ)
- 选择函数类型为 "极坐标 r = f(θ)"
- 输入表达式,例如:
  - `2*cos(theta)` (圆)
  - `theta` (阿基米德螺线)
- 点击 **绘制函数**

### 2. 参数控制

当函数包含参数时(非常数变量),系统会自动生成滑块控件:

**示例**: 输入 `A*sin(omega*x + phi)`
- 自动生成三个滑块: A, omega, phi
- 拖动滑块实时更新函数图象
- 每个滑块显示当前数值(精确到小数点后两位)
- 滑块范围: -10 到 10,步长 0.01

**常用参数化函数**:
- `A*sin(omega*x + phi)`: 正弦函数(振幅、角频率、相位)
- `a*x^2 + b*x + c`: 二次函数(系数)
- `a*exp(b*x)`: 指数函数(系数)
- `a/(x-b) + c`: 反比例函数(平移和缩放)

### 3. 图象变换

选择基础函数后,可以使用变换按钮快速修改图象:

#### 平移变换
- **← 左移**: 向左平移 1 个单位,`f(x)` → `f(x+1)`
- **右移 →**: 向右平移 1 个单位,`f(x)` → `f(x-1)`
- **↑ 上移**: 向上平移 1 个单位,`f(x)` → `f(x)+1`
- **下移 ↓**: 向下平移 1 个单位,`f(x)` → `f(x)-1`

#### 伸缩变换
- **X轴伸缩**: 水平拉伸 1.5 倍,`f(x)` → `f(x/1.5)`
- **Y轴伸缩**: 垂直拉伸 1.5 倍,`f(x)` → `1.5*f(x)`

#### 反射变换
- **X轴反射**: 关于 X 轴对称,`f(x)` → `-f(x)`
- **Y轴反射**: 关于 Y 轴对称,`f(x)` → `f(-x)`

### 4. 预设模板

从下拉菜单快速选择常见函数类型:

- **正弦函数 A*sin(ωx+φ)**: 研究三角函数参数影响
- **二次函数 ax²+bx+c**: 探索抛物线性质
- **指数函数 a*e^(bx)**: 观察指数增长/衰减
- **高斯函数**: 正态分布曲线
- **反比例函数 a/x**: 双曲线

选择模板后自动填充表达式并绘制。

## 高级技巧

### 组合变换

可以连续应用多个变换来创建复杂图象:

**示例**: 从 `sin(x)` 开始
1. 点击 "右移" → `sin(x-1)`
2. 点击 "Y轴伸缩" → `1.5*sin(x-1)`
3. 点击 "上移" → `1.5*sin(x-1)+1`

最终得到经过平移和缩放的 sine 曲线。

### 参数与变换结合

先使用参数滑块微调函数形状,再应用几何变换:

**示例**: `A*sin(omega*x)`
1. 调整 A=2, omega=0.5 (振幅加倍,周期加倍)
2. 点击 "左移" 2 次 → `2*sin(0.5*(x+2))`

### 对比不同函数

可以同时绘制多个函数进行对比:
1. 绘制 `x^2`
2. 再次输入 `(x-2)^2` 并绘制
3. 观察两条抛物线的位置关系

## 支持的数学函数和常量

### 基本运算
- `+`, `-`, `*`, `/`: 加减乘除
- `^` 或 `**`: 幂运算
- `sqrt()`: 平方根

### 三角函数
- `sin(x)`: 正弦
- `cos(x)`: 余弦
- `tan(x)`: 正切

### 指数和对数
- `exp(x)`: 自然指数 e^x
- `log(x)`: 自然对数 ln(x)

### 其他函数
- `abs(x)`: 绝对值

### 常量
- `pi`: 圆周率 π ≈ 3.14159
- `e`: 自然常数 e ≈ 2.71828

## 注意事项

1. **表达式语法**: 
   - 使用 `*` 表示乘法,不能省略(如 `2*x` 而非 `2x`)
   - 使用 `^` 或 `**` 表示幂运算
   - 函数调用需要括号,如 `sin(x)` 而非 `sin x`

2. **参数识别**:
   - 系统自动识别非常量字母为参数
   - 已知变量: `x`, `y`, `theta`
   - 已知函数: `sin`, `cos`, `tan`, `exp`, `log`, `sqrt`, `abs`
   - 其他字母视为参数并生成滑块

3. **性能考虑**:
   - 复杂函数可能需要更多计算时间
   - 建议每次绘制一个函数,避免过多重叠
   - 隐函数绘图较慢,请耐心等待

4. **错误处理**:
   - 如果表达式有误,会显示错误提示
   - 检查括号是否匹配
   - 确保使用正确的函数名

## 教学应用场景

### 1. 探究参数对函数图象的影响
- 使用 `A*sin(omega*x + phi)` 模板
- 分别调整 A、omega、phi
- 学生直观理解振幅、周期、相位的概念

### 2. 函数变换演示
- 从基础函数 `x^2` 开始
- 逐步应用平移、伸缩、反射
- 展示 `y=f(x)` 到 `y=af(bx+c)+d` 的变换过程

### 3. 比较不同类型函数
- 同时绘制线性、二次、指数函数
- 观察增长速度差异
- 理解函数渐近行为

### 4. 解析几何可视化
- 使用隐函数绘制圆锥曲线
- `x^2/a^2 + y^2/b^2 - 1` (椭圆)
- `x^2/a^2 - y^2/b^2 - 1` (双曲线)
- 调整参数 a、b 观察形状变化

## 故障排除

### 问题: 函数无法绘制
**解决**:
- 检查表达式语法是否正确
- 确认使用了正确的函数名(小写)
- 尝试简化表达式,分步测试

### 问题: 滑块不响应
**解决**:
- 确认表达式中包含参数
- 检查参数名是否与已知变量冲突
- 重新输入表达式并绘制

### 问题: 图象显示不完整
**解决**:
- 使用画布的缩放和平移功能
- 调整函数的定义域范围(需修改代码)
- 检查函数是否在可视范围内有定义
